You borrow a neighbor's ladder to change a porch bulb. When the bulb works, you set the ladder beside your garage. You plan to take it back after dinner. Then rain starts, and the ladder stays where you left it.

If you have time, carry it back when you finish. If you need to clean mud from its feet, do that first. Send a short message and ask where the owner wants it.

If you can't return it that day, say so. Give a time you can keep. Put the ladder by the door or add the return to your calendar. The lender knows when to expect it, and you have a clear next step.

Sometimes the owner will tell you to keep it longer. They may be away or may not need it yet. Follow that plan and return it when you agreed.

## Finish the Return

  <blockquote>Set the time. Bring the thing back.</blockquote>

</figure>

Use the same process for a book, casserole dish, key, phone charger, or anything else you borrow. Keep the item with the parts that came with it. Make a return plan before it disappears into a drawer, car, or closet.

![Three panels show a ladder being borrowed, used to change a porch bulb, and returned to hooks.](/images/essays/return/section-1.png)

*Borrow it. Use it. Return it.*

Check the item before you leave. Bring the case, cord, lid, feet, or other parts. If you can't find a part, tell the owner before you arrive. Return the item in the condition you received it when you can. If the item is dirty, clean it. If it is scratched, say so. If it is broken, tell the owner and offer to repair or replace it. Keep the explanation short and give the owner a clear choice.

If the owner asks before you return it, answer directly: "I have it. I can bring it at six." Then follow through. If six changes, send another message before six.

If you can't return the item yourself, ask where to leave it. Use a porch, shed, or another spot only after the owner agrees. Send a message once the item is there.

You don't need a long speech. Say when you will return the item, then do it. Once it is back, the task is finished. The ladder goes on its hooks. You go home.
