The shirt is the most traditional item in the release: one blank, one color direction, and a small size run that can be fulfilled without a full ecommerce backend.

## Sizing

This first run is available in `S`, `M`, `L`, and `XL`. Each size has its own checkout path so stock can be controlled manually.

## Release notes

The shirt is kept intentionally narrow for launch. One treatment, one colorway, and a short size list is easier to pack accurately and restock without guesswork.
