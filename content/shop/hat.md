The hat is the quietest item in the first release: one adjustable cap meant to carry the imprint without turning into a billboard.

## Fit and use

This release uses a single adjustable style so inventory stays simple and the page can stay honest about what is actually in stock.

## Release notes

The first run stays intentionally small. If inventory dries up, the page copy will be updated and checkout will be disabled until more hats are ready.
